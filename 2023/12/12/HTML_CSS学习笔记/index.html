文本元素

字体样式元素

  1. <b></b>：加粗文本。被该标签包围的文本会加粗显示。
  2. <big></big>：使文本变大。这个标签会使被它包围的文本字体大小变大。但是，请注意，这个标签在HTML5中已经被废弃，不推荐使用。
  3. <i></i>：斜体文本。被这个标签包围的文本会以斜体形式显示。
  4. <small></small>：使文本变小。这个标签会使被它包围的文本字体大小变小。但是，请注意，这个标签在HTML5中已经被废弃，不推荐使用。
  5. <tt></tt>：打字机文本。这个标签会使被它包围的文本以打字机文本的样式显示，通常用于表示打字机打印的文本。
  6. <sup></sup>：上标。这个标签会使被它包围的文本显示为上标，通常用于数学符号或化学方程式中的分数。
  7. <sub></sub>：下标。这个标签会使被它包围的文本显示为下标，通常用于化学方程式中的化学元素符号或数学中的幂运算。

cellpadding和cellspacing是HTML表格中使用的属性，用于控制表格单元格的样式和布局。

cellpadding是指单元格边框到内容之间的距离（留白），通常以像素为单位进行设置。例如，如果您设置cellpadding为10像素，那么单元格边框与内容之间的空白距离为10像素。

而cellspacing是指单元格之间的空白量，也就是表格中各个单元格之间的距离，同样通常以像素为单位进行设置。通过调整cellspacing可以改变表格中单元格之间的间距，从而影响整个表格的外观和布局。

需要注意的是，HTML规范中已经不建议使用cellspacing和cellpadding属性，因为它们已经被CSS样式所取代。在现代的网页开发中，建议使用CSS来设置表格的样式和布局，包括单元格间距和填充等。
